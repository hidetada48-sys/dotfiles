## プロジェクトの全体像

### 何を作ったか
XのブックマークをPlaywright（ブラウザ自動操作）で取得し、テキストファイルとしてGoogle Drive経由でNotebookLMに同期するツール。Claude Code のSkillとして発動できる。

### 開発の流れ（大まかな時系列）
1. **3/11** test2リポジトリ作成・Skillsの学習・構築開始
2. **3/12** Xブックマーク取得スクリプトの初期開発（Chromiumデバッグ・セッション保存）
3. **3/15** 大規模改善（X記事対応・引用ツイート・記事本文取得・パス修正など）
4. **3/16** dotfilesによる環境整備・SKILL.md整備
5. **3/17** PC間共有対応・スクロール取りこぼし修正・引用X記事・外部リンク対応

### 苦労したポイント（記事のネタ）
- XはAPIなしでのスクレイピングが非常に困難（セッション管理・DOM構造・仮想スクロール）
- X記事・通常ツイート・引用ツイート・外部リンクと取得パターンが多様
- DOM仮想化によるスクロール取りこぼし問題
- PC間で処理済みリストを共有する仕組みの構築
