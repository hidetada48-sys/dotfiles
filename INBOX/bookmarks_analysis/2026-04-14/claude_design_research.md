# Claude Design 調査まとめ

- **作成日：** 2026-04-21
- **情報源：** Anthropic公式 + Xブックマーク（2026-04-14〜04-20）

---

## 1. Claude Design とは

**Anthropic Labs** が2026年4月17日にリリースした実験的デザインツール。  
チャットでClaudeに話しかけるだけで、スライド・プロトタイプ・ワンページャーなどの視覚的成果物を生成できる。

> "make prototypes, slides, and one-pagers by talking to Claude."  
> — Anthropic公式アナウンス

**Anthropic Labsとは：** Instagram共同創設者 Mike Krieger が率いる、Claudeの最前線機能を実験・育成するチーム。Claude Designはそこから生まれた最初の大型プロダクト。

---

## 2. 主な機能（公式情報）

### 入力形式（何を渡せるか）

| 入力 | 内容 |
|---|---|
| テキスト | 日本語でも話しかけるだけ |
| 画像 | スクリーンショット・参考デザイン |
| ドキュメント | DOCX・PPTX・XLSX |
| Webキャプチャ | URLを渡してサイトから読み込み |

### 出力・エクスポート

| 出力先 | 内容 |
|---|---|
| **Canva** | 1クリックでエクスポート、そのまま編集可 |
| **PDF** | 資料として保存 |
| **PPTX** | PowerPoint形式で保存 |
| **スタンドアロンHTML** | そのまま公開できるWeb形式 |
| **Claude Code** | ハンドオフバンドルとして渡し、実装・リリースまで一気通貫 |

### 調整・コラボ機能

- インラインコメントで細かく指示
- 直接編集（テキストや要素を自分で変更）
- カスタムスライダーで微調整
- 組織内共有（リンク共有・編集権限付与）

### デザインシステム統合

- チームのコードベースとデザインファイルをオンボーディング時に読み込み
- 色・タイポグラフィ・コンポーネントを自動適用
- 以降のプロジェクトは全て統一スタイルで生成

## 5. 実用上の最重要情報：DESIGN.mdリポジトリ

### これは何か

GitHubに公開されているデザイン仕様集。Apple・楽天・LINE・MUJIなど**有名企業のブランドルール（色・フォント・余白・スタイル）をMarkdownで書き起こしたファイル**が収録されている。

### なぜ重要か

Claude Designにこのファイルを添付して使うだけで、**プロレベルのクオリティに一気に引き上げられる**。デザインの専門知識ゼロでも有名企業の「型」を借りられる。

```
活用ステップ：
1. DESIGN.mdリポジトリから好みのブランドのファイルを取得
2. Claude Designで作業開始時にファイルを添付
3. 「〇〇のデザインスタイルで〜を作って」と指示
4. 後はベースを好みで改変するだけ
```

> 「添付して使うだけでプロレベルのクオリティになる。後はベースを好みで改変するだけでOK。」  
> — @tetumemo

### 自分専用のDESIGN.mdを作る応用

自分のブランドカラーや好みのルールを書いたDESIGN.mdを一度作れば、毎回同じスタイルで出力させることが可能。

---

## 6. ワークフロー全体像

```
① Claude Designでデザイン（話しかけるだけ）
        ↓
② 微調整（インラインコメント / スライダー / 直接編集）
        ↓
③ エクスポート（Canva で仕上げ / PDF / PPTX）
        ↓
④ Claude Code にハンドオフ → Web アプリとしてリリース
```

## 7. 試すなら：推奨アクション

| 優先度 | アクション | 理由 |
|---|---|---|
| **今すぐ** | スライド1枚を試作する | 入口として一番わかりやすい |
| **今すぐ** | DESIGN.mdリポジトリを手元に用意する | 添付するだけでクオリティが跳ね上がる |
| **今後** | 自分のMYDESIGN.mdを作成する | 毎回同じスタイルで生成できるようになる |
| **今後** | Claude Code連携（ハンドオフ）を試す | デザイン→実装の一気通貫フローを体験 |
