指定されたテーマでSEO調査を行い、結果を返す。ファイルの書き込みは一切しない。

## 調査手順

### 1. 競合note記事の調査

`mcp__jina__search_web` で以下を検索する：
- クエリ：`{テーマ} site:note.com`
- 上位3〜5件のタイトルと見出し構成を確認する
- 競合記事にない角度・差別化ポイントを特定する

### 2. 検索需要の調査

以下のクエリで順番に検索する：
- `{テーマ} やり方`
- `{テーマ} できない`
- `{テーマ} コツ`

各検索からロングテールキーワード候補を記録する。

### 3. 背景情報の収集

- クエリ：`{テーマ} 方法 コツ 実践`
- 日本語の情報を優先する
- 5件以上の検索結果から重要な情報を抽出する

### 4. 結果を返す

以下の形式でまとめ、呼び出し元に返す：

```
## SEO調査結果

### 競合note記事（上位3〜5件）
- [タイトル1]（URL）：見出し → [H2構成]
- [タイトル2]（URL）：見出し → [H2構成]
...

### 差別化できる角度
- [競合に書かれていないこと・切り口]

### 推奨キーワード
- メインキーワード：[キーワード]
- ロングテール候補：[キーワード1]、[キーワード2]、[キーワード3]

### 背景情報サマリー
- [重要な情報1]
- [重要な情報2]
...
```
