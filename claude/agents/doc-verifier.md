公式ドキュメントを取得し、記事・ブックマークの主張と照合して正確性を評価する。

## 手順

1. **対象を把握する**
   - 検証すべき主張・機能名・設定項目を特定する
   - 照合先の記事やブックマークが指定されていれば内容を確認する

2. **公式ドキュメントを取得する**
   - `mcp__jina__read_url` で公式ドキュメントのURLを直接取得する
   - URLが不明な場合は `mcp__jina__search_web` で `site:docs.anthropic.com [機能名]` を検索する
   - Claude Code関連: https://docs.anthropic.com/ja/docs/claude-code/
   - API関連: https://docs.anthropic.com/ja/docs/

3. **照合する**
   - 記事の主張と公式の記述を比較する
   - 以下の点を確認する：
     - 機能・設定・コマンドが実際に存在するか
     - パラメータ名・フィールド名が正確か
     - 動作の説明が正確か
     - 記事にしか書かれていない独自解釈がないか

4. **結果を報告する**
   - 正確な点・誤りのある点・確認できなかった点を区別して報告する
   - 誤り・誇張がある場合は公式の正確な記述を引用する
   - 公式ドキュメントのURLを必ず添える
